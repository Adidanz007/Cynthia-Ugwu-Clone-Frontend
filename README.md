# Cynthia Ugwu - Product Designer Portfolio

A modern portfolio website showcasing the work of Cynthia Ugwu, a product designer based in Toronto.

## Features

- Smooth scrolling animations using Locomotive Scroll
- GSAP animations for engaging interactions
- Responsive design with modern UI
- Project showcase section
- About section
- Footer with social links

## Project Includes

- **The Plug** (2022)
- **ixperience** (2022)
- **hudu** (2022)

## Technologies Used

- HTML5
- CSS3
- JavaScript
- GSAP (GreenSock Animation Platform)
- Locomotive Scroll
- Remixicon (Icon library)

## Getting Started

1. Clone the repository
2. Open `project3.html` in your web browser
3. No build process required - it's ready to use!

## Files

- `project3.html` - Main HTML file
- `style3.css` - Main stylesheet
- `loco.css` - Locomotive scroll styles
- `script.js` - JavaScript animations and interactions
- Images: `cynthia.png`, `plug.png`, `ixperience.png`, `hudu.png`
